# TICKYET — Landing Page

Premium landing page for TICKYET, a digital growth agency. Built with Next.js 15 (App Router), TypeScript, Tailwind CSS, shadcn/ui patterns, and Framer Motion.

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Current milestone (1 of N)

This pass covers: project setup, folder structure, the sticky Navbar, and the Hero section. See the project status report from Claude for what's next.

## Brand tokens

Colors, type roles, and copy live in `src/lib/site-config.ts` and `tailwind.config.ts` — update them there rather than hard-coding values in components.
